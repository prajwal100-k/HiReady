This is the Phase 1 scaffold for the Hiready backend (Node.js + TypeScript + Express + Prisma).
It implements authentication (signup/login/session) and connects to PostgreSQL via Prisma.

## Quickstart (local)
1. Copy `.env.example` to `.env` and fill values.
2. `npm install`
3. `npx prisma migrate dev --name init`
4. `npm run dev`

## Notes
- This is scaffold code. You'll still need to `npm install` and run Prisma migrations.
